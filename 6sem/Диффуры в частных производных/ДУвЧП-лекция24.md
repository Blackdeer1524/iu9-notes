## 3. Функция Грина для уравнения гиперболического типа
$$\begin{cases}
\rho u_{tt} = div (k grad u) - qu + f, (M,t) \in Q_\infty\\
u(M,0) = \varphi(M)\\
u_t(M,0) = \psi(M)\\
\alpha(P) \frac{dU}{dn} + \beta(P)U = \mu(P,t), P \in S, t \in [0, \infty], |\alpha| + |\beta| \neq 0
\end{cases} \tag{*}$$
Определение
КЛассическим решением задачи $(*)$ называется функция u(M,t) непрерывная вместе с первыми производными в замкнутом цилиндре $\overline Q_\infty = \overline D \times[0, \infty]$
имеющая непрерывные производные второго порядка в открытом цилиндре $Q_\infty$, удовлетворяющая в этом открытом цилиндре уравнению
$$\rho u_{tt} = div (k\space  grad \space u) - qu + f$$
начальным условиям
$$u(M,0) = \varphi(M)\\
u_t(M,0) = \psi(M)$$
и граничным условиям(последнее выражение в $(*)$

**Отметим**: для существования классического решения необходимо(но не достаточно), выполнение условия согласования начальных и граничных условий:
$$\alpha \frac{d\varphi}{dn} + \beta \varphi |_{p \in S} = \mu$$
и 
$$\alpha \frac{d\psi}{dn} + \beta \psi |_{p \in S} = \mu$$

В силу линейности мы можем представить решение в виде(определения решений будут позже)
$$u(M,t) = u_1(M,t) + u_2(M,t) + u_3(M,t)$$

**Теорема 1(единственности)**
Задача $(*)$ может иметь только 1 классическое решение.

**Теорема 2(об устойчивости)**
Решение задачи $(*)$ устойчиво по правой части и начальным данным.

Теперь перейдем к конкртеному решению задачи.

### 3.1. Вынужденные колебания открытой струны. Функция Грина.

$$\begin{cases}
u_{tt} = a^2 u_{xx} + f(x,t), (x,t) \in \Omega \\
u(x,0) = 0 \quad u_t(x,0) = 0\\
u(0,t) = 0, \quad u_t(l,t) = 0, \\
x \in [0,l], t \in [0, \infty]
\end{cases} \tag{**}$$

Метод разделения переменных
$$u = X * T$$
$$T''X  = a^2 X'' T + f(x,t)$$
$$\frac{T''}{a^2 T} = \frac{X''}{X} = -\lambda$$
На практике $a^2$ нужно перекидывать в X.

I задача штурма-лиувиля
$$\begin{cases}
X'' + \lambda X = 0, 0 \leq x \leq l \\
X(0) = X(l) = 0
\end{cases}$$

1) $\lambda = 0$

$$X'' = 0 \implies X = Ax + B \implies \text{ только тривиальное решение}$$
2) $\lambda = -w^2 < 0$
$$X'' - w^2 X = 0 \implies X = C_1 e^{wx} + C_2 e^{-wx}$$
$$\begin{cases}
0 = C_1 + C_2\\
0 = C_1 e^{wl} + C_2 e^{-wl}\\
\end{cases}$$
Имеем только нулевое решение. отбрасыаем

3) $\lambda = w^2 > 0$
$$X'' + w^2 X = 0 \implies X = C_1 \cos wx + C_2 \sin wx$$
ГУ
$$\begin{cases}
0 = C_1 + C_2 * \\
0 = C_1 \cos wl + C_2 \sin wl
\end{cases}$$

Имеем корни при $w_n = \frac{\pi n}{l}, n = 1.. \infty$
$$\lambda_n = [\frac{\pi n}{l}]^2$$
$$X_n = \sin \frac{\pi n}{l} x$$
$$\lVert X_n \rVert^2 = l/2$$

Составим общее решение задачи
II Общее решение 
$$U(x,t) = \sum_{n=1}^\infty T_n(t) \sin \frac{\pi n}{l}x$$
где
$$T_n(t) = \frac{2}{l} \int_0^l u(\xi, t) \sin \frac{\pi n}{l}\xi d\xi$$по определению

III уравнение на функцию
Подставим $u(x,t)$ в исходное уравнение
$$u_{tt} = a^2 u_{xx} + f(x,t)$$
$$\sum T_n'' \sin\frac{\pi n}{l }x = a^2 \sum T -(\frac{\pi n}{l})^2 \sin \frac{\pi n}{l}x T_n(t) + f(x,t)$$
$$\sum [T_n'' + (a\frac{\pi n }{l})^2T_n] \sin \frac{\pi n }{l}x = f(x,t)$$
Воспользуемся условием ортогональности собственных функций
$$\int_0^l \sin \frac{\pi n}{l} x \sin \frac{\pi m}{l}x dx= \frac{l}{2} \delta_{nk}$$
$$T_k'' + (a\frac{\pi k }{l})^2T_k] = \underbrace{\frac{2}{l}\int_0^lf(x,t) \sin \frac{\pi k}{l}x dx}_{=f_k}$$
Получаем уравнение на T.
$$T_n'' + (a\frac{\pi k}{l})^2 T_n = f_n, t \in (0, \infty)$$
Добавим сразу начальных условий
Исходные однородные начальные условия преобразуем в(в силу однородности)
$$T(0) = 0, T'(0) = 0$$

Найдем решение

однородный случай
$$T_n '' + \frac{\pi n a}{l}^2 T_n = 0$$
Его решение
$$T_n = C_{1n} \cos \frac{\pi n a}{l}t + C_{2n} \sin \frac{\pi na}{l}t $$

неоднородны случай с помощью метода Лагранжа
$$T_n = C_{1n}(t) \cos \frac{\pi n a}{l}t  +C_{2n}(t) \sin \frac{\pi n a}{l}t$$
имеем
$$C_{1n}'\cos ... + C_{2n}' \sin ... - \frac{\pi na}{l}C_{1n} \sin ... + c_{2n} \frac{\pi na}{l}\cos ... = 0$$

По методу лагранжа
$$C_{1n}' \cos \frac{\pi na}{l}t + C_{2n}' \sin \frac{\pi na}{l}t = 0$$

Берем еще одну производную
$$T_n'' = - C_{1n}' \frac{\pi n a}{l} \sin ... + C_{2n}' \frac{\pi n a}{l} \cos ... - [\frac{\pi n a}{l}]^2 \left\{C_{1n} \cos ... + C_{2n} \sin ...\right\}$$
Подставляем в начальное уравнение и получаем
$$- C_{1n}' \frac{\pi n a}{l} \sin ... + C_{2n}' \frac{\pi n a}{l} \cos ... = f_n(t)$$
Получили систему на $C_{1n}', C_{2n}'$
$$\begin{cases}
C_{1n}' \cos \frac{\pi n a}{l}t + C_{2n}' \sin \frac{\pi n a}{l}t = 0\\
- C_{1n}' \frac{\pi n a}{l} \sin ... + C_{2n}' \frac{\pi n a}{l} \cos ... = f_n(t)
\end{cases}$$

$$-C_{1n}' = C_{2n}' \frac{\sin (\pi n at)/l}{\cos (\pi n at)/l}$$
Подставляем
$$\frac{\pi n a}{l} C_{2n}' \frac{\sin \frac{\pi n a}{l}t \sin \frac{\pi n a}{l}t + \cos ... \cos ... }{\cos ...}  = f_n(t)$$
$$\frac{\pi n a}{l}C_{2n}' \frac{1}{\cos \frac{\pi n a}{l}t} = f_n(t)$$
$$C_{2n}' = \frac{l}{\pi n a} f_n(t) \cos \frac{\pi n a}{l}t$$
$$C_{1n}' = -\frac{l}{\pi n a} f_n(t) \sin \frac{\pi n a}{l}t$$

Получили
$$\begin{cases}
C_{2n} = \frac{l}{\pi n a} \int_0^t f_n(\tau) \cos \frac{\pi n a}{l}\tau d\tau \\
C_{1n} = -\frac{l}{\pi n a} \int_0^t f_n(\tau) \sin \frac{\pi n a}{l}\tau d\tau  
 \end{cases}$$
Решение для T
$$T_n = -\frac{l}{\pi n a} \left\{\int_0^t f_n \sin ... \cos ... d\tau - \int_0^t f_n \cos ... \sin ... d\tau\right\}$$
$$T_n = \frac{l}{\pi n a} \int_0^t f_n \sin \frac{\pi n a}{l} (t - \tau)d\tau$$
решение задачи Коши для функции $T_n(t)$
IV решение 
$$u(x,t) = \sum_{n=1}^\infty T_n \sin \frac{\pi n}{l} x =$$
$$= \sum_{n=1}^\infty  \frac{l}{\pi n a} \int_0^t f_n \sin \frac{\pi n a}{l} (t - \tau)d\tau \sin \frac{\pi n}{l}x$$
Учтем значение 
$$f_n = \frac{2}{l}\int_0^l f(\xi,\tau) \sin \frac{\pi n }{l} \xi d\xi$$

Получаем
$$u(x,t)  = \int_0^t d\tau \int_0^l d\xi f(\xi,\tau) \underbrace{\left\{ \sum_{n=1}^\infty \frac{2}{\pi n a} \sin \frac{\pi}{l}\xi \sin \frac{\pi n}{l}x\sin\frac{\pi n a}{l}(t-\tau)\right\}}_{=G}$$

$$G(x,\xi, t -\tau) = \sum_{n=1}^\infty \frac{2}{\pi n a} \sin \frac{\pi}{l}\xi \sin \frac{\pi n}{l}x\sin\frac{\pi n a}{l}(t-\tau)$$
Классическое решение задачи $(**)$.

Определение:
Функция
$$G(x,\xi, t) = \sum_{n=1}^\infty \frac{2}{\pi n a} \sin \frac{\pi}{l}\xi \sin \frac{\pi n}{l}x\sin\frac{\pi n a}{l}t$$
называется функцией Грина или функцией влияния мгновенного импульса. 

Функция Грина с физической точки зрения обозначает отклонение точки струны с координатой x в момент времени t при возбуждении струны в начальный момент времени $t = 0$ мгновенным точечным импульсом в точки $x_0$. 
В общем случае имеет следующий вид.
$$G(x,\xi,t) = \sum_{n=1}^\infty \frac{X_n(x) X_n(\xi) \sin a \sqrt{\lambda_n} t}{ \lVert X_n\rVert^2 a \sqrt{\lambda_n}}$$

### 3.2. Формула Грина для уравнения колебаний.
решение в ограниченной области через функцию Грина.

функция Грина $G(M_0, M, t_0 -t)$ - решение начально-краевой задачи.

$$\begin{cases}
\rho G_{tt} = L_m G + \delta(M, M_0)\delta(t_0-t) \\
\alpha \frac{dG}{dn} + \beta G|_S = 0 \\
G|_{t\ge t_0} 0 & G_t|_{t \ge t_0} = 0
\end{cases} \tag{3}$$

Функция грина полученная при решении этой задачи будет иметь вид
$$G(M_0, M, t_0 -t)  = \sum_{n=1}^\infty \frac{\sin a\sqrt{\lambda_n}(t_0 - t)}{a\sqrt{\lambda_n}} v_n(M) v_n(M_0)$$
Где
$\lambda_n$ - собственные значения, $v_n$ - ортонормированные собственные функции и они являются решениями задачи Штурма-лиувиля для оператора L
$$\begin{cases}
Lv + \lambda \rho v = 0\\
...|_S = 0
\end{cases}$$

Мы хотим решить задачу
$$\begin{cases}
\rho u_{tt} = L u + f(M,t) \\
\alpha \frac{du}{dn} + \beta u|_s = \mu(P,t) \\
u|_{t=0} = \varphi & u_t|_{t=0} = \psi
\end{cases}$$

Чтобы решить такую задачу рассмотрим 2 уравнения.
$$\rho u_{tt} = L u + f(M,t)$$
$$\rho G_{tt}= Lu + \delta(M,M_0) \delta(t_0 -t)$$

Умножим первое на функцию грина, а второе на u
$$\begin{cases}
\rho u_{tt}G = G Lu + G *f \\
\rho u G_{tt} = u LG + u \delta(M,M_0) \delta(t_0 -t)
\end{cases}$$
Вычтем из одного другое
$$\rho \left(G u_{tt} - u G_{tt}\right) = \left(GLu - uLG\right) + \left(G f + u \delta(M,M_0) \delta(t_0 - t)\right)$$

Интегрируем
$$\int_0^\infty \int (G u_tt - u G_tt) dv dt = \int \int (GLu - uLG) dv dt + \int \int (Gf - u \delta \delta)dv dt$$

Рассмотрим куски уравнения отдельно
$$\int_0^\infty  (G u_{tt} - u G_{tt})dt = $$
$$= \int_0^\infty u_{tt} G \eta(t_0 - t)dt - \int_0^\infty u G_{tt} \eta(t_0 -t)dt = $$
$$= \int_0^{t_0} (u_{tt} G - u G_{tt}) dt = $$
Вычислим по частям
$$ = u_t G|_0^{t_0} - \cancel{\int_0^{t_0} u_t G_{t} dt} - u G|_{0}^{t_0} + \cancel{\int_0^{t_0} u_t G_t dt} = $$
$$= u_t G|_{0}^{t_0} -u G_t |_0^{t_0} = -G(M_0, M, t_0) u_t(0) + u(M, 0) G_t(M_0, M, t_0)$$

Рассмотрим второй кусок
$$\int_0^\infty \int_D\underbrace{(GLu - uLG)dvdt}_{\text{вторая формула грина}}  + \int_0^\infty \int_D Gf dvdt - \underbrace{\int_0^\infty \int_D u \delta \delta dv dt}_{\text{свойства дельта функции}} =$$
$$ = \int_0^\infty dt \oint_S ds (G\frac{du}{dn_p} - u \frac{dG}{dn_p}) + \int_0^t \int_D fG dv dt - u(M_0, t_0) = $$
Подставляем
$$\int_0^\infty [u(M, 0) G_t(M_0, M, t_0) - u_t G(M_0, M, t_0)]dt = \int_0^\infty \oint_S ds (G \frac{du}{dn_p} - u \frac{dG}{dn_p})dsdt + \int_0^{t_0} G f dt - u(M_0, t_0)$$

утем, что 
$$\frac{dG}{dt}|_{t=0} = - \frac{d}{}$$

Получаем
$$u(M_0, t_0) = \int_0^t \int_D G(M_0, M, t_0 - t)f(M,t) dvdt + \int_0^{t_0} dt\oint_Sds (G(M_0, P, t_0 - t) \frac{du}{dn} - u(P,t) \frac{dG}{dn}) + \int_D \rho(G(M_0, M, t_0)u_t(M,0) + u(M,0) + G_t(M_0, M, t_0))dv$$

### 3.3 Решение начально-краевой задачи для уравнения колебаний в ограниченной области

3 типа задач: ГУ дирихле, ГУ неймана и третья

Задача Дирихле $\alpha = 0, \beta = 1$
$$u|_s = \mu \qquad G|_s = 0$$
$$u(M,t) = \int_0^t \int_D G(M,Q, t-\tau)f(Q,t)dv dt - \int_0^t \oint_S R \mu \frac{dG}{dn} ds dt + \int_D \rho (G(M,Q,t) \psi(Q) + \varphi G_t)dv$$

Задача Неймана $\alpha = 1, \beta = 0$
$$\frac{du}{dn}|_s = \mu \qquad \frac{dG}{dn}|_s = 0$$
$$u(M,t) = \int_0^t \int_D G(M,Q,t-\tau)f(Q,t)dvdt + \int_0^t \oint_S kG(M,P,t-\tau)mu(P,\tau)dvdt + \int_D \rho(G \psi + \varphi G_t(M,q,t))$$

третья краевая задача $\alpha=1, \beta = h$
$$\frac{du}{dn} + uh|_S = \mu \qquad \frac{dG}{dn} + hG|_s = 0$$

$$u(M,t) = \int_0^t \int_D G(M,Q,t-\tau)f(Q,\tau) dv dt + \int_0^t \oint_S k(P)G(M, P, t-\tau)\mu(P,t) ds dt + \int_D \rho(\psi G + \varphi G_t) dv$$
