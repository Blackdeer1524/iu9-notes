## причина 1 - погрешность округления
Рассмотрим пример
$$\begin{cases}
-10^{-7}x_1 + x_2 = 1\\
x_1 + 2 x_2 = 4
\end{cases}$$

Способ 1
$$\begin{cases}
x_1 = 10^{7}x_2 - 10^{7} \\
x_1 + 2x_2 = 4
\end{cases}$$
$$x_2 = 2 - \frac12(10^7x_2 - 10^7)$$
$$x_2 = \underbrace{1.000000}_{\text{сохраняет ЭВМ}}\cancel2$$
тогда
$$x_1 = 10^7x_2 -10^7 = \textcolor{red}{0}$$
из-за округления получили ошибку вычислений

Способ 2
$$\begin{cases}
x_2 = 1 + 10^{-7}x_1\\
2x_2 = 4 - x_1
\end{cases}$$
$$4-x_1 - 2 -2 * 10^{-7}x_1 = 0$$
$$2 = 1.0000002 x_1$$
$$x_1 \approx 2$$
тогда
$$x_2 =1 + 10^{-7}*2 \approx 1$$
Тогда оба уравнения в системе примерно равны

ВЫВОД: от способа решения зависит результат вычислений

Рассмотрим математический аппарат исследования влияния ошибок
Пусть $\varphi(x,y)$
$\tilde x = x \pm \Delta x$
$x$ - точное значение
$\Delta x$ - абсолютная ошибка измерения
$\tilde x$ - значение с погрешностью

Абсолютная погрешность вычислений функционала $\varphi(x,y)$
$$\Delta \varphi \leq |\varphi(\tilde x, \tilde y) - \varphi(x,y)| \le |\frac{\partial \varphi}{\partial x} dx + \frac{\partial \varphi}{\partial y} dy| \le$$
$$\le |\frac{\partial \varphi}{\partial x}| |\Delta x| + |\frac{\partial \varphi}{\partial y}| |\Delta y|$$

Вывод:
$$\delta = \left| \frac{\Delta \varphi}{\varphi}\right| \le \frac{1}{|\varphi|} (|\frac{\partial \varphi}{\partial x}| |\Delta x| + |\frac{\partial \varphi}{\partial y}| |\Delta y|)$$
Частные производные расчитываем численно

Пример применения формулы
$\varphi = (\sqrt 2 -1)^6$
$\varphi(x,y) = (x-y)^6$, тогда 
$$\delta_\varphi \le \frac{1}{|\varphi|} (|\frac{\partial \varphi}{\partial x}| |\Delta x| + |\frac{\partial \varphi}{\partial y}| |\Delta y|) = \frac{1}{(x-y)^6} (6|x-y|^5 \Delta x + 6|x-y|^5\Delta y) = $$
$$=6\frac{\Delta x + \Delta y}{|x-y|}$$
для данного примера
$$\Delta x < \sqrt 2 - \frac75 = 0.0142$$
$$\Delta y = 0 \Leftarrow y=\tilde y = 1$$
тогда
$$\delta_\varphi \le 6\frac{0.0142}{0.5} \approx 0.207$$
тогда
$$\delta_\varphi = 20\%$$

# 3. Источники возникновения погрешностей
источники делятся на 
- деление на малые числа
- вычитания больших, близких друг к другу чисел

Определение
Норма X - $\lVert X \rVert$ с следующими условиями:
$\lVert x \rVert > 0$
$\lVert x \rVert = 0 \Leftrightarrow x = 0$
и так далее


Примеры норм
$$\lVert X \rVert = (\sum |x_i|^p)^{\frac1p}$$
Определение нормированного пространства

Определение
Норма квадратной матрицы - число, $\lVert A \rVert$ и удовлетворяет свойствам
1) все как и для нормы
2) $\lVert AB \rVert \le \lVert A \rVert \lVert B \rVert$

Определение
норма матрицы A **согласованна** с нормой вектора $\lVert x \rVert$, если 
$$\lVert Ax \rVert \le \lVert A \rVert \lVert x \rVert$$

Замечание
Использование согласованных норм позволит далее получать требуемые оценки для погрешностей различных методов, в том числе итерационных. 

Определение
Норма матрицы A **подчинена** норме вектора $\lVert x \rVert$, если норма А вводится следующим образом:
$$\lVert A \rVert = \sup_{x \neq 0} \frac{\lVert Ax\rVert}{\lVert x\rVert} = \sup_{\lVert x \rVert =1} \lVert Ax \rVert$$
Замечание
подчиненные нормы согласованы с соответствующей ей нормой.
док-во
$$\frac{\lVert Ax \rVert}{\lVert x\rVert } \le \sup \lVert A x \rVert = \lVert A \rVert \qquad\square$$

Источники погрешностей как упоминалось формируется и из неточности измерений или ошибки исходных данных.

Пусть есть СЛАУ 
$$Ax = f \tag{1}$$
Искаженное СЛАУ
$$(A + \delta A)(x + \delta x) = f + \delta f$$
Тогда
$$\cancel{Ax} + A\delta x + \delta Ax + \delta A \delta x = \cancel{f} + \delta f \tag 2$$
$$A \delta x + \underbrace{\delta A \delta x}_{\to 0} = \delta f - \delta A x$$
пренебрегаем по порядку величины, тогда получаем
$$A \delta x = \delta f - \delta A x$$
$$A^{-1}A \delta x = A^{-1}\delta f - A^{-1}\delta A x$$
$$\delta x = A^{-1}(\delta f - \delta A x)$$
Пусть выберем *согласованную* норму, тогда
> Чаще мы будем использовать равномерную норму, для евклидовой будут более сложные вычисления

$$\lVert \delta x \rVert = \lVert A^{-1} (\delta f - \delta Ax)\rVert \le \lVert A^{-1}\rVert \lVert \delta f - \delta A x \rVert \le$$
$$\le \lVert A^{-1}\rVert (\frac{\lVert f \rVert}{\lVert f \rVert}\lVert \delta f \rVert + \frac{\lVert A \rVert}{\lVert A\rVert}\lVert \delta A \rVert \lVert x\rVert)$$
т.к. $\lVert f \rVert = \lVert Ax\rVert \le \lVert A\rVert \lVert x\rVert$, то
$$\lVert \delta x\rVert \le \lVert A^{-1}\rVert (\lVert A\rVert \lVert x\rVert \frac{\lVert \delta f\rVert}{\lVert f\rVert} + \lVert A\rVert \lVert x\rVert \frac{\lVert \delta A \rVert}{\lVert A\rVert}) =$$
$$= \lVert x\rVert \mu_A (\underbrace{\frac{\lVert \delta f\rVert }{\lVert f\rVert}}_{\delta_f} + \underbrace{\frac{\lVert \delta A\rVert}{\lVert A \rVert}}_{\delta_A})$$
тогда мы получаем, что 
$$\delta_x \le \mu_A(\delta_f + \delta_A)$$
$\mu_A = \lVert A\rVert \lVert A^{-1}\rVert$ - число обусловленности матрицы А. 

Замечание
$\mu_A$ показывает устойчивость СЛАУ к возмущениям. Чем меньше тем лучше. 
1) $1<\mu_A<10$ ошибки входа слабо сказываются на вычислениях или СЛАУ устойчиво
2) $\mu_A \ge 10^3$ - СЛАУ не устойчиво к возмущениям
3) в промежутке нужно оценивать отдельно

Приведем пример реального расчета
Рассмотрим СЛАУ
$$\begin{cases}
100x_1 + 99x_2 = 199 \\
99x_1 + 98x_2 = 197
\end{cases}$$
Точное решение $x=(1,1)^T$
Исказим правую часть
$$\begin{cases}
100x_1 + 99x_2 = 198.99 \\
99x_1 + 98x_2 = 197.01
\end{cases}$$
тогда решение будет следующим $x = (2.97, -0.97)^T$

Найдем число обусловленности матрицы
$$\mu_A = \lVert A\rVert \lVert A^{-1}\rVert $$
Выберем согласованную норму - равномерную 
$$\lVert x \rVert = \max_{i} x_{i}$$
$$\lVert A \rVert = \max_i \sum_j |a_{ij}|$$
тогда 
$$\begin{matrix}
\lVert f \rVert = 199 \\
\lVert \delta f \rVert = 0.01 \implies\\
\frac{\lVert\delta f\rVert}{\lVert f\rVert} = 0.5*10^{-4}\\
\lVert A \rVert = 199 \\
A^{-1} = \begin{pmatrix} -98 & 99 \\ 99 & -100\end{pmatrix} \\
\lVert A^{-1}\rVert = 199 \implies
\mu_A = 199^2 \le 4*10^4
\end{matrix}$$
тогда
$$\delta_x \le 4*10^4 * 0.5*10^{-4} \le 200\%$$
что мы и наблюдали в вычислениях.

## Погрешность вычислений
$$\frac{\lVert \delta x\rVert}{\lVert x \rVert} \approx \mu_A n * g(A) * p^{-t}$$
где
$\mu_A$ - число обусловленности матрицы
$n$ - размер матрицы
$p$ - основание машинной арифметики $p=2$
$t$ - число значащих цифр
$g(A)$ - коэфициент роста ошибки, определяется как 
$$g(A) = \max_{k,i,j} \left( \frac{\max _{i,j} |a_{ij}^{(k)}|}{\max |a_{ij}|} \right)$$
где k -номер шага прямого матода исключений(например метод Гаусса)
$a_{ij}^{(k)}$ - состояние матрицы на kм шаге
$a_{ij}$ - состояние исходной матрицы
