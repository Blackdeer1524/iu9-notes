# 9.6. Интегральное представление функции Бесселя
$$J_\nu(x) = \sum_{k=0}^\infty \frac{(-1)^k}{\Gamma(k+1)\Gamma(k+\nu+1)} \left(\frac x2\right)^{2k+\nu}$$
Представление функции Бесселя в виде ряда.
Учтем формулу 
$$\frac 1{\Gamma(z+1)} = \frac{e^{i\pi z}}{2\pi i}\int_\gamma e^{-t}t^{-z-1}dt$$
$$\Gamma(k) = k!, \text{ т.к. k целое}$$
Получим
$$J_\nu = \sum_{k=0}^\infty \frac{(-1)^k}{k!} \frac{e^{i\pi(k+\nu)}}{2\pi i} \int_\gamma e^{-t} t^{-k - \nu -1}dt \left(\frac{x}{2t}\right)^{2k+ \nu} =$$
$$= \frac{e^{i\pi\nu}}{2\pi i} \int_\gamma e^{-t} (\frac{x}{2})^\nu \left[\sum_{k=0}^\infty \frac{(-1)^k}{k!} \frac{e^{i\pi k}}{t^k} (\frac{x^2}{2})^k\right] \frac{dt}{t}=$$
Учтем, что $e^{i\pi k} = (-1)^k$
$$= \frac{e^{i\pi\nu}}{2\pi i} \int_\gamma e^{-t} (\frac{x}{2t})^\nu \underbrace{\left[\sum_{k=0}^\infty \frac{1}{k!}  (\frac{x^2}{4t})^k\right]}_{= e^{x^2/4t}} \frac{dt}{t}=$$
$$= \frac{e^{i\pi\nu}}{2\pi i} \int_\gamma e^{-t} \left(\frac x{2t}\right)^\nu e^{x^2/4t} \frac{dt}{t}=$$
$$=\frac{e^{i\pi\nu}}{2\pi i} \int_\gamma \left(\frac x{2t}\right)^\nu e^{x^2/(4t) - t} \frac{dt}{t}$$

В качестве контура $\gamma$ выберем контур гамма-функции
==Картинка==
Сделаем замену переменных $t\to\xi$
$$t = \frac{x}{2}e^{-i(\xi-\pi)}$$
$$\frac{x}{2t} = e^{i(\xi - \pi)}$$
$$dt = \frac x2 \underbrace{e^{-i(\xi-\pi)}}_{=2t/x}(-id\xi) = -it/td\xi = -i d\xi$$
