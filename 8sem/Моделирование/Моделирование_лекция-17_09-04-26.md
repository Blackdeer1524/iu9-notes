# Дисперсионный анализ
Сам по себе дисперсионный анализ - развитие на многомерный случай метода наименших кавадратов. 

>[!note] Определение
>Метод наименьших квадратов и стандартизированный набор правил для проверки статистических гипотез называется дисперсионным анализом.

Термин предложил Фишер, Фишер работал на Родждельской агробиологической станции математиком. Что он делал? Его пригласили для решения практической задачи, в Англии специфичный климат, урожай могут собрать только 1 и скудный, места не много, поэтому нужно было выделить 10 делянок. Агробиологов интересовали мутации гороха. Есть желтый горох, он сушится, а есть зеленый - он засыхает. Зеленый горох - отрицательная мутация, и не хотелось, что бы они размножались. Он решил ее в частном случае и для ее решения предложил метод дисперсионного анализа. 

Согласно этому методу полное изменение числового вектора $(y_1 ... y_n)^T$ разбивается на составляющие, связанные с результатом действия различных классификационных факторов. Факторы фиксируются индексами данных в формулах.

В методе наименьших квадратов мы минимизировали сумму квадратичных отклонений от среднего
$$\sum_{i=1}^n(y_i - \overline y)^2 \to \min$$
Эта формула показывает изменчивость данных. При этом изменчивость данных может зависит от различных факторов.

> Пример из истории. В МГТУ когда-то давно увлекались статистическими моделями. Тогда считали активный запас времени. Идея была такая - студенты живут на разных расстояниях. Кто-то живет в общаге рядом с институтом, кто-то в Измайлово, кто-то живет на крайней Москве, кто-то за МКАДом. Поэтому студенты находятся в разных условиях. Второй момент - студенты перед экзаменом по разному спят. В общаге может быть шумно, а нужно готовиться к экзамену, а кто-то изолировано живет. Был опрос по разным характеристикам. Дальше анализировали какие факторы влияют на оценку экзамена.
> Фактор дороги мало влияют на успеваемость студентов. Но сильно влияли факторы сна в сессию - один из главных факторов. Но как они определили?

Сейчас запишем общую формулу, которая не практична, но показывает структуру метода. Одна большая сумма разделяется на несколько. На практике считается по-другому. Для факторов A B
$$\sum_{ij} (y_{ij} - y_{* *})^2 = \sum_{ij}(y_{i*} - y_{**})^2 + \sum_{ij}(y_{* j} - y_{**})^2 + \sum_{ij}(y_{i*} - y_{* j} + y_{**})^2$$
>Почему таблица двумерная? Пусть фактор A - сон студента, B - время затраченная на дорогу и целевой признак $y$. И для каждого студента есть трио значений. Тогда $i$ - индекс соответствующий строке фактора $A$, $j$ - индекс соответствующий строке фактора $B$. Если факторов большее число, то число индексов большее.
>Почему не $\overline y$, а $y_{**}$? в МНК все просто, а здесь нужно собирать значения по факторам.
>  - Первая сумма - основной вклад фактора A
>  - Вторая сумма - основной вклад фактора B
>  - Третья сумма - эффект взаимодействия A и B или совместный вклад факторов A и B

Как посчитать? Третья сумма - вычитание других

Данные представляются в форме матрицы из столбцов соответствующих фактору A, фактору B и значению функции $y$. Строки матрицы содержат данные для каждого объекта выборки. Точки - результаты осреднения по соответствующим индексам:
- $y_{i*}$ - значение из столбца фактора A
- $y_{* j}$ - значение из столбца B
- $y_{**}$ - осреднение значений из столбца $y$

Совместный вклад факторов A и B не является суммой или разностью их влияния, а рассматривается как дополнение общего разброса с учетом основных факторов A и B. Здесь нельзя его показать как шум, так как совместный вклад может значительно превышать отдельные вклады.

Могут быть все комбинации размера величин вкладов:
- если большой вклад дают отдельно A или B, то можно какой-то из признаков может не влиять. 
- Если большой вклад A и совместный A и B (или B), то тогда можно говорить о колигации факторов A и B. фактором B не пренебрегаем, но учитываем в совместном появлении с A
- Если отдельные вклады A и B малы, а совместный вклад большой, то можно говорить об ассоциации факторов. Всегда с одним есть другой. 

Такой анализ дал вывести, что схема лекция+домашка дает лучшие результаты и схема 2 на 3: 1 знакомство, 1 повторение, 1 закрепление. 

> Формализация метода может являться статистической моделью

Путем анализа данных с помощью такой статистической модели можно сравнивать среднеквадратические значения. Чем больше величина среднеквадратического значения, тем более сильным является влияние фактора. Среднеквадратичная погрешность интерпретируемая часто как шум, показывает совместный вклад факторов. Чаще всего рассматривает однофакторный анализ и двуфакторный анализ.

---
Напрямую предыдущую формулу нельзя использовать. Для практического использования перейдем к другой таблице от ABy к 

|     | B1  | B2                          | ... | Bm                          |
| --- | --- | --------------------------- | --- | --------------------------- |
| A1  |     | $y_{12k}, k = 1 ... K_{12}$ |     |                             |
| A2  |     |                             |     |                             |
| ... |     |                             |     |                             |
| An  |     |                             |     | $y_{nmk}, k = 1 ... K_{nm}$ |

где A1 ... An - разбиение значений из столбца A на n диапазонов. Аналогично для B1 ... Bm. В каждой ячейке будет размещаться подвыборка значений исходной выборки, параметры которых лежат в соответствующем диапазоне значений.
Тогда $y_{i*}$ рассматривается как усреднение влияния $i$-го A фактора, а $y_{* j}$ как усреднение влияния $j$-го B фактора, $y_{**}$ - усреднение всех значений.


В ряде случае число значений в каждой клетке разнится, кроме того данные в опыте могут поступать последовательно (обогащение данных), в силу чего суммы квадратов оказываются взвешенными. На практике рассматривают сопутствующую задачу, дисперсию (слева) раскладывают на 2 суммы квадратов, первая из которых соответствует сумму квадратов, характеризующую изменчивость по строкам для фактора A, и по столбцам. Остатки соответствуют общую изменчивость. В идеале - равен 0, но на практике не встречается, будет трактоваться как межвыборочная сумма квадратов.  Если сравниваемые выборки на самом деле не отличаются, то считаются что появившиеся различия связаны с случайными флуктуациями. В этом случае значимым оказывается последний член суммы. Если имеются различия в выборках на уровне матожидания, но отличаются на некоторый эпсилон. То эта величина оказывается взвешенной с учетом размерностей выборок. 

| Различия между выборками | $\frac{mn}{m+n}(\overline x_1 - \overline x_2)^2$                                                   | 1 степеней свободы     |
| ------------------------ | --------------------------------------------------------------------------------------------------- | ---------------------- |
| Различия внутри выборок  | $\sum_{i=1}^m(x_{1i} - \overline x_1)^2 + \sum_{j=1}^n(x_{2j}- \overline x_2)^2$                    | m+n-2 степеней свободы |
| Полная изменчивость      | $\sum_{k=1}^{nm} (y_i - \overline y)^2$ или $\sum_{i=1}^n\sum_{j=1}^m(y_{ij} - \overline y_{**})^2$ | m+n-1 степеней свободы |

масштабируем на степени свободы. Будем считать, что была одна выборка и появилась другая выборка. Была $x_1^1, x_1^2 ... x_1^m$ и добавилась $x_2^1 ... x_2^m$. Совмещая получаем 
$$y_1 = x_1^1 ... y_m = x_1^m, y_{m+1} = x_2^1 ... y_{m+n} = x_2^n$$

У нас есть среднее по $x_1$ и среднее по $x_2$ и главная средняя - среднее по $y$. среднее по y определяется по формуле. Эту величину будем называть главным средним.
$$\overline y = \frac{1}{m+n} \sum_{i=1}^{m+n}y_i = \frac{m\overline x_1 + n\overline x_2}{m+n}$$
Откуда вытекает алгебраическое тождество
$$\sum_{k=1}^{m+n} (y_i - \overline y)^2 = m\sum_{i=1}^m (x_1^i-\overline x_1)^2 + n\sum_{j=1}^n (x_2^j-\overline x_2)^2 + \frac{mn}{m+n}(\overline x_1 - \overline x_2)^2$$
Последнее слагаемое трактуется как межвыборочная сумма квадратов если выборки принадлежат одной генеральной совокупности должно равняться нулю.
