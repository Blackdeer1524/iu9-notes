**Метод Свенна** находит отрезок, в котором скорее всего будет минимум функции(он может не найти), поэтому нужно проверить корректность. Для этого, функция должна быть **унимодальной**.
Есть 2 теоремы для проверки функции на унимодальность:
1. $f'(x)$ не убывает на отрезке
2. $f''(x) \ge 0$ (если существует) на отрезке

Дальше с помощью метода Фиббоначи нужно найти точку минимума функции. ВСЕ производные должны быть не захардкожены, и их нужно считать численно с помощью сетки:
$$f'(x) \approx \begin{cases}
\text{1й вариант } \frac{f(x+h) - f(x)}{h} \text{ - правая разность}\\
\text{2й вариант } \frac{f(x) - f(x-h)}{h} \text{ - левая разность}\\
\text{3й вариант } \frac{f(x+h) - f(x-h)}{2h} \text{ - центральная разность, самая точная}\\
\end{cases}$$
Для поиска второй производной численно
$$f''(x) \approx \frac{f(x-h) - 2f(x) + f(x+h)}{h^2}$$
