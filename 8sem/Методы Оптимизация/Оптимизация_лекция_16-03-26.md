На лабе нужно реализовать метод внутренней точки, простой версии

Рассматриваем методы условной оптимизации, до этого была безусловная оптимизация, при этом методы условной оптимизации сводим к безусловной оптимизации, или по-другому рассмотрим как можно найти приблизительное решение.

Решаем следущую задачу условной оптимизации. Задача:
находим минимум $f(x). x\in \mathbb R^n$
при условии(из набора)
- $h_i(x) = 0$
- $g_j(x) \le 0$

постановка
$$\arg \min \begin{cases}
f(x), x \in\mathbb R^n \\
h_i(x) = 0 \\
g_j(x) \le 0
\end{cases}\tag1$$
h,g - функции системы ограничений, желательно выпуклые функции


**Метод штрафных функций** - строим новую целевую функцию на базе системы, при этом функции системы ограничений добавляют штраф, либо к бесконечности уходит, либо просто больше 0.

$$Q(x,r) = f(x) + \sum_{i=1}^m r_i H_i[h_i(x)] + \sum_{j=m+1}^n r_j G_j(g_j(x)) \tag2$$
где $H_i$ и $G_j$ устремляются к бесконечности при выходе за пределы, $x \in \mathbb R^n$ и $r = (r_1, ... r_m, r_{m+1}, ... r^n)$

после построения необходимо решить задачу безусловной оптимизации

Замечание: приближенное решение системы 1 находится путем решения последовательности 2 для $r^{(l)}\to\infty$ стремящиеся к бесконечности.  

Замечание: метод штрафных функций заключается в том, что выбираются функции $H_i, G_i$ так, чтобы они становились отличными от 0 при нарушении ограничений их задаваемых. Другими словами при нарушении ограничений вспомогательная функция $Q(x, r)$ возрастает. Или по-другому эти функции генерируют штрафы вне ограниченной области.

Рассмотрим 2 способа: метод внешней точки и метод внутренней точки.

Пусть X - допустимая область $h_j(x)=0 \text{ \& }g_j(x) \le0$, при этом глобальный минимум $f(x)$ вне области $X$.

**Замечание для метода внешней точки**: начальная точка релаксационной последовательности может быть выбрана как внутри области ограничений, так и снаружи. Как правило выбирают начальную точку снаружи области ограничений. в методе внешней точки барьерные функции внутри области равны 0 или очень малы, а вне просто большое значение.  

**Замечание для метода внутренней точки:** начальная точка релаксационной последовательности может быть выбрана только внутри и никак иначе. В методе внутренней точки внутри области значнея граничной функции должны стремиться к 0, а вне - стремится к бесконечности. 

## **О выборе функций штрафов**
### о $h_j(x)=0$
$$H_j(h_j(x)) : H_j(h_j(x)) \to 0, h_j(x) \to 0$$
Примеры:
-  $H_j(a) = |a|$
- $H_j(a) = a^{2n}$
### $g_j(x)\le 0$
$$G_j(g_j(x)) : \begin{cases}G_j(g_j(x)) = 0, g_j(x) \le 0 \\
G_j(g_j(x)) \to \infty g_j(x) > 0
\end{cases}$$
Примеры:
- $G_j(a) = \frac12 (a + |a|) = u(a)$
- $G_j(a) = u^{2n}(a)$
- $G_j(a) = -\ln(a)$
- $G_j(a) = -1/a$

## Пример оптимизации внутренней точки
Замечание: везде используем поиск минимума численным поиском с использованием одномерных методов оптимизации

Постановка задачи: 
$$\arg \min \begin{cases}
f(x) = x^2 , x\in X\\
X = \{x-2 \ge 0\} \subset \mathbb R
\end{cases}$$
Приведем постановку к необходимому виду
$$\arg \min \begin{cases}
f(x) = x^2 \\
2-x \le 0
\end{cases}$$
Выберем для метода внутренней точки функцию $G_j(a) = \frac12(a + |a|)$
$$Q(x,r) = f(x) + \frac{r}{2} (2-x + |2-x|)$$
Для данной функции выполняется
$$\lim_{r\to\infty} \min_{x \in \mathbb R} Q(x,r) = \min_{x\in X} f(x)$$

Можно выбрать другую штрафную функцию и тогда получим
$$Q(x,r) = x^2 + \frac{1}{r(2-x)}$$
После начинаем шаги итеарции:
1. Берем $r=1$
2. ищем точку минимума при первом шаге
3. Берем $r=2$ и снова минимизируем из предыдущей точки
4. Продолжаем до момента пока шаг не станет меньше эпсилон
