# Анализ издержек производства
## Расчетный метод нахождения точки безубыточности
$$Выручка = Ц_{ед}*X$$
$$S_{общ} = S_{пост} + (S_{уд. перем}*X)$$
$$Ц_{ед}*X = S_{пост} + (S_{уд.перем}*X)$$
выражаем X
$$X = \frac{S_{пост}}{Ц_{ед} - S_{уд.перем}}$$
### Задача
$Ц_{ед} = 98 руб/ед$
$X = 2000 ед$
$S_{ед} = 92 руб/ед$
$S_{пост} = 126 тыс. руб$

Надо найти выручку, S_{пер}, S_{уд. пост}, S_{уд. пер.}, S_{общ}, прибыль, прибыль на единицу, точку безубыточности

$Выручка = Ц_{ед}*X = 196 тыс. руб$
себестоимость = 184 тысяч рублей

Прибыль = 196 - 184 = 12 тыс рублей

$S_переменные = 184-126 = 58 тыс$

$S_{уд. пост} = 126 тыс / 2 тыс = 63 руб/ед$
$S_{уд. пер.} = 58/2 = 29 руб/ед$

Прибыль на единицу = 6 руб/ед

точка безубыточности = 
$$\frac{126 тыс}{98 - 29} = 1826.08 = 1827$$
