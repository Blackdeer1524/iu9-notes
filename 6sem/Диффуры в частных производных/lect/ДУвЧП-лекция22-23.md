## 2.4.3 Решение начальной задачи для неоднородного уравнения теплопроводности в трехмерном просранстве с помощью фундаментального решения

Рассмотрим постановку задачи. Пометим ее как (4.3.)
$$\begin{cases}
u_t = a^2 \Delta u + f(M,t), (M,t)\in\Omega_3 \\
u(M,0) = \varphi(M), M \in \mathbb R^3
\end{cases} \tag{4.3.}$$
При этом
$$\Omega_3 = \mathbb R^3 \times (0,T]$$
$$\overline \Omega_3 = \mathbb R^3 \times [0,T]$$
Определение: Фундаментальное решение $G(M,Q,t)$ задачи Коши для уравнения теплопроводности $u_t = a^2 \Delta u$ - это такое решение в облсти $\Omega_3$, которое
1) удовлетворяет начальному условию: $G(M,Q,0) = \delta(M,Q)$, причем $M,Q \in \mathbb R^3$
2) Непрерывно всюду на $\overline \Omega_3$, кроме точки $(Q,0)$ т.е. при $M \neq Q, t \neq 0$

Построим фундаментальное решение
Для этого рассмотрим лемму

**Лемма**
Если в задачи Коши
$$\begin{cases}
u_t = a^2 \Delta u\quad (M,t) \in \Omega_3\\
u(M,0) = \varphi(M), M \in \mathbb R^3
\end{cases} \tag{4.4}$$
с начальной функцией $\varphi = \varphi_1(x) \varphi_2(y) \varphi_3(z)$, то
решение задачи (4.4) является функцией
$$u(M,t) = u_1(x,t) u_2(y,t) u_3(z,t)$$
где
$u_1(x,t)$ решение одномерной задачи Коши:
$$\begin{cases}
u_{1t} = a^2 u_{1xx} &(x,t) \in \Omega\\
u_1(x,0) = \varphi_1(x) & x \in \mathbb R
\end{cases}$$
Для других функций аналогично, только другие переменные и индексы.

Без док-ва хотя оно тривиально.

В трехмерном случае $\delta$-функцию можно представить по определению
$$\delta(M,Q) = \delta(x,\xi) \delta(y,\eta) \delta(z,\zeta), \quad M(x,y,z), Q(\xi,\eta,\zeta)$$

Применим эту Лемму к нашей задаче.
$$\begin{cases}
u_t = a^2 \Delta u\\
u(M,0) = \varphi(M)
\end{cases}$$
И мы хотим найти функцию $G(M,Q,t)$, которая будет удовлетворять условию $G(M,Q,0) = \delta(M,Q)$
Получим по лемме
$$G(M,Q,t) = G(x,\xi,t) G(y,\eta, t) G(z,\zeta,t)$$
Учитывая, что явный вид функции G мы знаем
$$G(x,\xi,t) = \frac{1}{2a\sqrt{\pi t}} e^{-\frac{(x-\xi)^2}{4a^2 t}}$$
Поэтому получаем следующее решени
$$G(M,Q,t) = \left(\frac{1}{2a\sqrt{\pi t}}\right)^3 e^{- [(x-\xi)^2 + (y-\eta)^2 + (z-\zeta)^2]/(4a^2 t)}$$
Получили ***фундаментальное решение для уравнения теплопроводности в трехмерном пространстве*** или ее также называют функцией влияния мгновенного точечного теплового источника.

Отметим, что она также является симметричной и мы можем источник и наблюдатель менять местами
$$G(M,Q,t) = G(Q,M,t)$$
Теперь мы можем записать решение задачи Коши (4.3). Мы говорили, что его решение можно представить как 
$$U(M,t) = U_1(M,t) + U_2(M,t)$$
Для каждого из них строим отдельное решение. Построим для $U_1$
$$\begin{cases}
u_t = a^2 \Delta u\\
U(M,0) = \varphi(M)
\end{cases}$$
Для $U_2$
$$\begin{cases}
u_t = a^2 \Delta u + f(M,t)\\
U(M,0) = 0\end{cases}$$
Запишем решение с помощью фундаментального решения
$$U_1 = \iiint_{\mathbb R^3} G(M,Q, t) \varphi(Q) dV_Q$$
Первое - интеграл Пуассона
$$U_2 = \int_0^t \iiint_{\mathbb R^3} G(M,Q,\tau) f(Q,\tau) dV_Q d\tau$$

В итоге получаем следующее общее решение
$$U = \iiint_{\mathbb R^3} G(M,Q, t) \varphi(Q) dV_Q + \int_0^t \iiint_{\mathbb R^3} G(M,Q,t-\tau) f(Q,\tau) dV_Q d\tau$$

## 2.5. Функция Грина для начально-краевой задачи для уравнения теплопроводности
### 2.5.1 Формула Грина для уравнения теплопроводности
Получим аналог третьей формулы Грина для уравнения теплопроводности. Для этого нам понадобится фундаментальное решение.
$$G_0(M,Q,t) = \left(\frac{1}{2a\sqrt{\pi (t-\tau)}}\right)^3 e^{- [(x-\xi)^2 + (y-\eta)^2 + (z-\zeta)^2]/(4a^2 (t-\tau))} \eta(t-\tau)$$
Укоротим формулу введя расстояние между двумя точками
$$R_{MQ} = \sqrt{(x-\xi)^2 + (y-\eta)^2 + (z-\zeta)^2}$$
$$G_0(M,Q,t) = \left(\frac{1}{2a\sqrt{\pi (t-\tau)}}\right)^3 e^{- R_{MQ}^2/(4a^2 (t-\tau))} \eta(t-\tau)$$
В пункте 4.2. было показано, что эта функция по переменным $Q,\tau$ удовлетворяет уравнению 
$$-\frac{dG_0}{d\tau} = a^2 \Delta_Q G_0(M,Q) + \delta(M,Q) \delta(t-\tau)$$
Пусть $u(Q,\tau)$ - решение неоднородного уравнения, тогда
$$\frac{du}{d\tau} = a^2 \Delta u + f(Q,t), (Q,t) \in \Omega_3$$
Подставим
$$\begin{cases}
-\frac{dG_0}{d\tau} = a^2 \Delta_Q G_0(M,Q) + \delta(M,Q) \delta(t-\tau) \left| \times u(Q,t) \right.\\
\frac{du}{d\tau} = a^2 \Delta u + f(Q,t) \left| G_0(M,Q,t-\tau) \right.
\end{cases}$$
$$\begin{cases}
-\frac{dG_0}{d\tau} U(Q,t)= a^2 U(Q,t)\Delta_Q G_0(M,Q) + \delta(M,Q) \delta(t-\tau) U(Q,t)\\
\frac{du}{d\tau}G_0 = a^2 G_0\Delta u + f(Q,t)G_0
\end{cases}$$
Из первого вычтем второе
$$-\frac{dG_0}{d\tau} U(Q,t) - G_0 \frac{du}{d\tau} = a^2\left[ U(Q,t)\Delta_Q G_0(M,Q)  - G_0 \Delta U\right] - G_0 f+ \delta(M,Q) \delta(t-\tau) U(Q,t)$$
Проинтегрируем это выражение по $\tau \in (0,\infty)$ и по всей области D

$$\int_0^\infty \int_D \left[G_0\frac{dU}{d\tau} + U \frac{dG_0}{d\tau}\right] dV d\tau = a^2 \int_0^\infty \int_D \left[U(Q,t)\Delta_Q G_0(M,Q)  - G_0 \Delta U\right] dVd\tau + \int_0^\infty \int_D \left[ G_0f - U\delta(M,Q) \delta(t-\tau)\right]dV d\tau$$
Учтем
$$\int_D G_0 \Delta u - u \delta G_0 dV = \text{вторая формула Грина} = \oint_S \left[G_0 \frac{dU}{d\eta} - U \frac{dG_0}{d\eta}\right] ds$$
$$\int_0^\infty \int_D u(Q,t) \delta(M,Q)\delta(t-\tau) dV dt = u(M,t)$$
$$G_0(M,Q,t-\tau) = 0, \text{при } \tau > t$$
В первом интеграле пределы изменятся по этому свойству
Получим
$$\int_0^t \int_D \left[G_0 \frac{du}{d\tau} + u \frac{dG_0}{d\tau}\right] dV d\tau = a^2 \int_0^t\oint_S \left[G_0 \frac{dU}{d\eta} + U \frac{dG_0}{d\eta}\right] ds dt + \int_0^t \int_D G_0f dV dt - u(M,t)$$

также учтем, что 
$$\int_0^t \left[G_0 \frac{du}{d\tau} + u \frac{dG_0}{d\tau}\right] d\tau = \int_0^t \frac{d}{d\tau}(G_0U) d\tau = \left.G_0 U \right|_0^t = G_0(M,Q,t-\tau)U(Q,t)|_0^t =$$
$$=\underbrace{ G_0(M,Q,t)U(Q,t)}_{\text{по определению} = 0} - G_0(M,Q,t)U(Q,0) = - G_0(M,Q,t) U(Q,0)$$
получим, с учетом выражения $[U(Q,0) = \varphi(Q)]$
$$- \int_D G_0(M,Q,t) U(Q,0) dV  = a^2 \int_0^t\oint_S \left[G_0 \frac{dU}{d\eta} + U \frac{dG_0}{d\eta}\right] ds dt + \int_0^t \int_D G_0f dV dt - u(M,t)$$
Выразим из выражения $u$
$$U(M,t) = \int_D G_0(M,Q,t) U(Q,0) dV  +a^2 \int_0^t\oint_S \left[G_0 \frac{dU}{d\eta} + U \frac{dG_0}{d\eta}\right] ds dt + \int_0^t \int_D G_0(M,Q,t-\tau)f(Q,t) dV dt$$
Мы получили ***Формулу Грина для уравнения теплопроводности***.
### 2.5.2.  Функция Грина для уравнения теплопроводности
Мы хотим с помощью формулы Грина решение начально-краевых задач
$$\begin{cases}
U_t = a^2 \Delta U + f(M,t)\\
U(M,0) = \varphi(M)\\
\alpha \frac{\partial U}{\partial \eta} + \beta U |_S = \mu(\rho, t)
\end{cases}$$
ДЛя этого необходимо, чтобы функция $G$ удовлетворяла нулевому граничному условию
$$\alpha \frac{\partial G}{\partial \eta} + \beta G |_S = 0$$
тогда 

Задача Дирихле $\alpha = 0, \beta = 1$
Тогда формула Грина
$$U(M,t) = \int_D G(M,Q,t)U(Q,0) dV - a^2 \int_0^t \oint_S U(P,\tau) \frac{dG(M,P,t-t\tau)}{dn_p} dS d\tau + \int_0^t \int_D G(M,Q,t-\tau) f(Q,t)dV d\tau$$

Задача Неймана $\alpha = 1,\beta = 0$
$$U(M,t) = \int_D G(M,Q,t)U(Q,0)dV + a^2\int_0^t \oint_S G(M,Q,t-\tau) \frac{dU}{dn_p} dS d\tau + \int_0^t \int_D G(M,Q, t-\tau) f(Q,t) dVd\tau$$
Третья краевая задача $\alpha = 1, \beta = h(p)$
$$U(M,t) = \int_D G(M,Q,t) U(Q,0) dV + a^2 \int_0^t \oint_S G(M,P,t-\tau)\left[\frac{dU}{dn_p} + h U\right]dS d\tau + \int_0^t \int_D G(M,Q,t-\tau) f(Q,t) dV d\tau$$

Определение: функция $G(M,Q,t-\tau)$ - функция Грина уравнения теплопроводности, если она удовлетворяет условиям:
1) Функция Грина представима в следующем виде $G(M,Q,t-\tau) = \frac{1}{2a\sqrt{\pi(t-\tau)}} e^{-R_{MQ}/(4a^2(t-\tau))} + v$, где $v$ - $v_t = a^2 \Delta v,\quad v|_{t=\tau} = 0$
2) удовлетворяет $\alpha \frac{dG}{d\eta} + \beta G|_S = 0 |\alpha| + |\beta| \neq 0$

функция Грина доопределяется в $t\leq \tau$ нулем.

Чтобы найти функцию Грина решим начально-краевую задачу:
$$\begin{cases}
\frac{\partial G}{\partial t} = a^2 \Delta G + \delta(M,Q)\delta(t-\tau)\\
\alpha \frac{dG}{d\eta} + \beta G|_S = 0 \quad|\alpha| + |\beta| \neq 0 \\
G(M,Q,t-\tau)|_{t\leq \tau} = 0
\end{cases}$$
