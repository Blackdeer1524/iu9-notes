# 13.1 Ссудный процент и его экономическая роль в условиях рынка
## 1. Сущность ссудного процента
**Ссудный процент** - плата получаемая кредитором от заемщика от пользования ссужеными деньгами или материальных ценностей (но чаще всего деньгами).

движение ссужаемой стоимости
$$Д \to Д', \text{ где } Д' = Д + \Delta d$$
$\Delta d$ - плата за кредит (ссудный процент).

> [!tip] возникновение ссудного процента
> появился за 2 тыс лет до н.э. имел форму натуральных ссуд. например, взяли в ссуду 1 кг зерна, вернули 1.5 кг. С развитием он принял только денежную форму.
### Необходимость ссудного процента
1. наличие у кредитора временно свободных денежных средств, передача которых в временное пользование другому субъекту связано с рисками. => процент - плата за риск.
2. ссудный процент необходим банку как коммерческая прибыль
3. ссудный процент уплачивается заемщиком для стимулирования рационального использования заемных средств и своевременного погашения ссуды.

### Функции ссудного процента
1. распределительная - связана с распределением вновь созданной стоимости ($\Delta d$)
2. сохранение ссудного капитала - ссудный процент является платой за риск отчуждения кредитором его временно свободных денежных средств.
3. регулирующая - ссудный процент регулирует соотношение между спросом и предложением заемных средств
4. посредством процента осуществляется регулирование объема привлекаемого в банк депозита

###  Формы процента

| Классификационный признак                       | Форма ссудного процента                                                                                                                                                                                                                                  |
| ----------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| По формам кредита                               | - Коммерческий процент - процент по коммерческим кредитам (одно предприятие дает взаймы другому)<br>- Банковский процент - займ, даваемый банковской организации.<br>- Государственный кредит - кредит, который дается вышестоящим органом нижестоящему. |
| По виду кредитных учреждений                    | - Учетный процент ЦБ РФ - ставка финансирования центральным банком<br>- банковский процент<br>- процент по операциям ломбардов(ставка небанковских организаций)                                                                                          |
| По видам инвестиций с привлечения кредита банка | - процент по кредитам в оборотные средства<br>- процент по инвестициям в основные фонды<br>                                                                                                                                                              |
| по срокам кредитования                          | процент по краткосрочным, среднесрочным и долгосрочным ссудам                                                                                                                                                                                            |
| по видам оперцаций кредитного учреждения        | депозитный процент                                                                                                                                                                                                                                       |
