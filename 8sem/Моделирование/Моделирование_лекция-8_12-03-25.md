# точные оценки параметров
> Пусть случайная величина $\xi: x, F(x,\theta)$ - функция распределения $f(x,\theta)$ - плотность распределения. Истинное значение $\theta$ неизвестно. Требуется оценить по результатам N повторов независимых результатов наблюдения значение параметра $\theta$. 

Совокупность распределений F(x,\theta) при разных значениях параметра $\theta$ называют параметрическим семейством распределений. Любая функция от результатов наблюдения, называется статистикой эксперимента или статистикой $\tau = \tau(x_1, ... x_n)$

Задача построения точной оценки параметра сводится к нахождению такой функции от результатов наблюдений. $\hat\theta = \hat\theta(x_1, ...x_n)$ в таком случае является статистикой, значение которого близко к значению параметра $\theta$. 

## свойства оценок
Оценка $\hat \theta$ называется **несмещенной** оценкой параметра $\theta$ если ее математическое ожидание совпадает с истинным значением параметра, то есть $M\hat\theta = \theta$.

*Пример*: Можем теоретически обосновать, что рождение мальчиков или девочек равновероятно. Но в реальных наблюдениях может в какие-то годы больше рождаться одних или других. Такая оценка статистикой будет асимптотически несмещенной, то есть при увеличении выборки будем лишь приближаться к истинному значению

Оценка $\hat\theta$ является **асимптотически несмещенной** оценкой $\theta$, если $\lim_{n\to N} M\hat\theta = \theta$, где N - объем генеральной совокупности, для схемы Бернулли $N=\infty$.

Объем выборки $n$ также влияет на состоятельность оценок. Оценка $\hat \theta_n$ называется **состоятельной** оценкой параметра $\theta$, если при $n\to\infty$ она сходится по вероятности к истинному значению параметра: $\forall \varepsilon > 0, \lim_{n\to\infty} P\{|\hat\theta_n - \theta| > \varepsilon\} = 0$

Справедлива **теорема**:
Пусть $\hat\theta_n$ --- асимптотически несмещенная оценка параметра $\theta$, при этом 
$$\lim_{n\to\infty} S^2(\hat\theta_n) = 0$$
то такая оценка является состоятельной. 

Пусть у нас 2 оценки параметра $\theta$ --- $\hat\theta^{(1)}, \hat\theta^{(2)}$.  Если
$$S^2_1 = M(\hat\theta^{(1)} - \theta)  \quad <\quad M(\hat\theta^{(2)} - \theta) = S^2_2$$
То оценка 1 является более эффективной чем второй.

# Доверительная оценка или доверительные интервалы
> Пусть $x_1, ... x_n$ - независимая случайная повторная выборка из генеральной совокупности. и так далее как в предыдущем пункте

Предположим, что для параметра $\theta$ построен интервал $\theta \in [\theta_-; \theta^-]: P\{\theta_- \le \theta \le \theta^-\} = \gamma$, тогда этот интервал называют **доверительным** для параметра $\theta$ с коэффициентом доверия $\gamma$ для всех возможных $\theta$.

Функция $\tau = \tau(x_1, ... x_n, \theta)$ называется **центральной статистикой**. На функцию накладываются определенные условия.

Пусть имеется статистика $\tau$. Пусть она описывается монотонно убывающей функцией. 

Квантилем уровня $\alpha$ функции распределения $F(t)$ называется величина $K(\alpha): F(K(\alpha)) = \alpha \Leftrightarrow P\{t \le K(\alpha)\} = \alpha$

Оставим зазор с обоих сторон функции распределения: $\alpha \in [\varepsilon_1; 1-\varepsilon_2]\implies t \in [t_1, t_2] : F(t_i) = \varepsilon_i$

Нам бы хотелось, чтобы статистика $P\{t_1 \le \tau(x_1, ... x_n) \le t_2\} = \gamma = 1-\varepsilon_1 - \varepsilon_2$
Если $\varepsilon_1=\varepsilon_2 = \varepsilon$, то $\gamma = 1-2\varepsilon \implies \varepsilon = (1 - \gamma)/2$
$$\begin{cases}
\tau(x_1, ... x_n, \theta_-) = t_1 \\
\tau(x_1, ... x_n, \theta^-) = t_2
\end{cases}$$
если исходная функция монотонно убывающая, то меняем t1t2 местами.
