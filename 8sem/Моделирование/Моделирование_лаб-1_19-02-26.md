# Лабораторная 1 - описание поверхности 
Сегодня построим статическую модель, в чем-то проще чем динамическая, так как не изменяется в времени. Нам нужно решить следующую задачу:

Задана матрица высот в таблице - псевдослучайная регулярная сетка. Нужно описать-приблизить данную поверхность. 

Каждому значению x,y соответствует одно значение высоты. псевдорегулярная сетка.

| y\x  | 0.12 | 0.23 | 0.46 |
| ---- | ---- | ---- | ---- |
| 1.19 |      |      |      |
| 2.13 |      |      |      |
| 2.28 |      |      |      |
Причем мы не определяем ни конкретные значения, ни определенный характер поведения.

В понедельник пары не будет. Но будет двойная пара. Тестовые данные у АБ. На первом часе выбираем метод решения. Потом в перерыве АБ пройдет и посмотрит за 15 минут. Вторая пара - реализация модели.

Моделирование не численные методы. Можно пользоваться готовыми библиотеками. Но данная лаба будет соревновательной, и важна точность приближения. Те у кого точность будет хорошей - выше в рейтинге. А в библиотечных реализациях нельзя часто управлять точностью. Поэтому рекомендуется реализовать самому. Сегодня можно прикидочное решение, но в течении недели - написать самому и улучшить результаты

1. Построение математической модели - аналитическая статическая модель. В перерыве - комментарии от АБ
2. Написание программы и тестирование на тестовых данных. Можно 20х20 или 10х10. И присылаем АБ материалы.
