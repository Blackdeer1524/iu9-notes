# Планирование эксперимента: основные понятия
Планирование эксперимента - комплекс мероприятий, направленный на эффективную постановку опыта.

Простой пример: 3 веса $a,b,c$, надо их измерить. и 
- $y_1$ - значение для $a$
- $y_2$ - значение для $b$
- $y_3$ - значение для $c$

Уточнить значение можно следующим образом $y_i' = y_i - y_0$. (Адьюстировка - настройка)
Поэтому число экспериментов увеличится на 1.

Из примера с весами видно, что увеличение точности может потребовать дополнительных экспериментов.

Основная цель планирования эксперимента - достижения максимальной точности измерения, при минимальном количестве опытов и при сохранении статистической достоверности результата. Получается в время планирование эксперимента решается оптимизационная задача. Планирование эксперимента применяется для поиска оптимальных условий развития процесса.

С этой целью оцениваются функции связи между параметрами опыта (модели), применяются критерии согласия для проверки статистических гипотез, методы многомерного статистического анализа как для аппроксимации данных, так и для уменьшения количества факторов. 

Рассматривают следующие типовые задачи теории планирования эксперимента:
1. Поиск значений параметров системы
2. Оценка функциональных связей между показателями системы по результатам проведенного эксперимента
3. Оценка влияния уровней параметров системы на показатели качества исследуемого объекта или системы реального мира

На основе этих задач решаются и частные задачи планирования эксперимента.
1. Оценки степени соответствия показателей качества образцов при проведении испытания заданным требованиям при минимальном объеме проведенного эксперимента
2. проведение отсеивающего эксперимента для выявления параметрами, которые незначительно влияющих на показатели качества системы
3. адаптивное планирование в условиях управления процессом при изменении условия функционирования системы или ее поведений. адаптивные - с обогащением данных.
4. и другие

Основной проблемой при моделировании являются неполные первоначальные знания характеристик исследуемого объекта. То есть решение задач ТПЭ предусматривает априорной информации об изучаемом объекте реального мира для определения оббщей последовательности планирования эксперимента, которая будет уточнена для обогащения данных. Тем самым достигается возможность рационального управления экспериментом. 

Процесс и объект можно рассматривать как черный яшик.
$$\overline X \xrightarrow{A(\overline V)} \overline Y$$
Переменные $\overline V$ - принятые управляемые факторы.

Задача ТПЭ снизить число управляемых факторов. При проведении эксперимента они меняются в заданном диапазоне и дискретно. Факторы в эксперименте бывают качественные и количественные. Количественные данные дискретизируются а качественные данные квантируются. Далее будем считать, что все факторы являются количественными и представлены непрерывными величинами, если другое не оговорено особо (непрерывны в рамках одного кванта). 

Переменным N можно сопоставить геометрическое понятие факторного пространства, координатные оси которого соответствуют значениям факторов. Совокупность конкретных значений всех факторов определяют точку в многомерном факторном пространстве на правах координат. На объект или систему могут влиять возмущающие факторы, но они являются случайными и управлению не поддаются. Пространство факторов задано таким образом, что
$$V_i^{\min} \le V_i \le V_i^{\max}, i = 1 ...n$$
Как правило факторы нормализуются, при нормализации значения факторов переводятся в безразмерным величинам. Представление эксперимента в виде гиперкуба позволяет определить центральную точку - центр эксперимента или центр плана.

> почему нормализация не предобработка, а этап планирования эксперимента? Нормализация - сжатие или растяжение между осями.

Переход из одного факторного пространства в другое (нормализованное), соответствует сжатию или растяжению пространства в направлении координатных осей и не влияет на разброс каких-то факторов вдоль этих осей. 

Различают Активный и пассивный эксперимент. При пассивном исследователь только регистрирует значения, при активном формирует условия в результате которого будут получены регистрируемые значения.

План эксперимента - совокупность данных, определяющих количество условия и порядок реализации опытов. 

> причем здесь порядок реализации? Сначала должны идти опыты с наиболее вероятным изменением результата. 

Конкретные значения факторов таким образом обозначаются $V_{ij}$ $i$ номер опыта $j$ номер фактора. Вся совокупность $V_{ij}$ образуют матрицу плана
$$\begin{pmatrix}
V_{11} & V_{12} & V_{13} \\
&... \\
V_{N1} & V_{N2} & V_{N3}
\end{pmatrix}$$
Различают полный и дробный план эксперимента
- Полный содержит все возможные комбинации факторов при определенном числе уровне. С ростом числа факторов таким образом растет по показательному закону. 
- Дробный план содержит часть комбинаций значений из полного плана. Строится дробный план в соответствии с критериями оптимальности плана.

## Свойства планов
Будем считать, что математическая модель в общем случае представляет собой функцию $y = f(x_1, ... x_n)$, $x_i$ - значения аргументов при которых стоят коэффициентов. В частном случае может быть описана линейной регрессией. Оценка коэффициентов функции при заданных значениях факторов считают решением задачи планирования экспериментов. В общем случае решают на основе метода наименьших квадратов. Если задача многомерна, то часто переходят к дисперсионному анализу (что очень похоже на метод наименьших квадратов(МНК)). Требуется минимизировать сумму квадратов отклонений от заданных значений. 
$$\frac{1}{N} \sum_i\left(y_i - f(x_1^{(i)}, ... x_n^{(i)})\right)^2 \to 0$$
Приравнивая к нулю частные производные от квадратичной формы по коэффициентам можно найти оптимум с помощью решения системы нелинейных уравнений. Решение возможно при линейной независимости базисных функций. Если это условие не выполнено, то коэффициенты взаимозависимы и полученные выражения для функции отклика можно рассматривать только как интерполяционную формулу. 

Что делать, чтоб обеспечить независимость? Формирую матрицу плана так, чтобы коэффициенты оставались взаимонезависимыми получаем значение коэффициентов имеющие разброс самые маленькие коэффициенты будут соответствовать признакам практически не вносящие вклад в функцию отклика. То есть задача заключается в определении общей формы записи функции отклика желательно в виде полинома.

Не всегда возможно из теоретических соображений выделить вид функции $f(x_1, ... x_n)$. Рекомендуется в универнсальном, удобном для практического применения виде, как правило в виде полинома в некотором ортонормированном базисе. Тогда системой базисных функций будет совокупность степенных функций с целыми неотрицательными значениями показателей степени и функция отклика окажется линейной относительно неизвестных коэффициентов. 

> Бывает путаница на экзамене. $y = b_0 + b_1 x + b_2 x^2 + b_n x^n$. Линейно от коэффициентов а не от переменных x.

Можно зафиксировать степень полинома, а не раскладывать в бесконечный ряд, хвост ряда считать слагаемым ошибкой. Степень полинома обычно задается исследователем априорно, но шанс ошибки исследователя есть, поэтому степень может уточняться в ходе исследования.

> Ученые и изобретатели - самые ленивые люди. Им не хочется считать.

Наибольшее распространения получают полиномы 1, 2 и 3 порядков. Коэффициенты полиномов принято называть эффектами факторов. Даже если выбранная модель нелинейна, ее можно упростить искусственно, переходя от нелинейной к линейной. Линеаризация применяется в методе наименьших квадратов. 

Таким образом применение ТПЭ основано на ряде допущений:
1. Функция отклика содержит в своем составе детерминированную (полиномиальная часть) и случайную составляющую (остаточная случайная часть, $\delta$ от разложения в ряд)
2. Факторы эксперимента $V_k$ измеряются с пренебрежимо малой ошибкой, по сравнению с ошибкой в определении величины отклика $y$. Учет помех экспериментального характера является трудно разрешимой проблемой. В то же время ошибка значимо пополняется из-за величины $\delta$.
3. Разделение ошибок между детерминированной и недетерминированной части модели. Снижение погрешности происходит из разделения ошибок
4. Дисперсия среднего значения функции отклика в различных точках равны друг другу. То есть выборочные оценки дисперсии должны быть однородны.

Указанные допущения позволяют использовать для расчета коэффициентов МНК, который дает эффективные и несмещенные оценки и сам метод прост в реализации.

>Для МНК не требуется соблюдение нормального распределения

Этот метод в любом случае дает решение, минимизирующее сумму квадратов отклонения отфуннкции отклика. В настоящее метода используют свыше 20 критериев оптимальности планов, среди которых выделяют 2 основные группы:
1. Критерии, связанные с ошибками коэффициентов и их ошибки
2. Критерии, связанные с ошибкой поверхности отклика (то что отправляетя в delta)

Критерии первой группы используются для задачи выделения доминирующих, более значимых параметров на начальных этапах. Геометрическое толкование свойтсв ошибок свзано с элипсоидами и их рассеивания. Пространственное расположение и форма элипсоида полностью зависят от плана эксперимента. 

Критерий D оптимальности - минимизация объема элипсоида рассеивания ошибок. 

... с минимальной суммарной дисперсией коэффициентов. 

Критерий E оптимальности соответсвует план в которой максимальная дисперсия коэффициентов окажется минимальной. 


Выбор критерия зависит от задачи исследования. Е критерий обычно применяют для изучения влияения отдельного фактора на поведение. если интересует задача оптимизации, то используют Д критерий, а А критерий для улучшение оптимального плана. 
Критерий второй группы используются для определения поверхности отклика. Основным критерием является критерий Ж оптимальности, позволяет построить план с минимальным значением максимальной ошибки в описании функции отклика. 

Выделяют 3 свойства оптимальных планов:
- свойства **ортогональности**. План ортогонален, если выполняется условие парной ортогональности столбцов таблицы планирования. При ортогональном планировании коэффициенты полинома определяются вне зависимости друг от друга вычеркиванием или добавлением слагаемых в функцию отклика. Для ортогональных планов эллипсоид рассеивания ориентирован в пространстве так, что направелние осей рассеивания совпадает с направлением осей параметров. 
- **рототабельности**. обеспечивает для любого направления эксперимента равназначность точности оценки функции отклика. Есть центр плана, ротоабельность делает окружность, по поверхности сферы - дисперсия одинаковая. или называется свойство униморфности. Выполнение такого требования целесообразно, когда исследователь не знает точное расположение области поверхности отклика с оптимальными значениями параметров. Рототабельный план гарантирует, что мы посчитаем значения параметров. Указанная область будет определена на основе модели полученной по результатам эксперимента.
- **композиционности**.  Предполагает построение линейной модели, а при обогащении данных наблюдений, переход к моделям второго порядка с использованием ранее полученных данных наблюдений. При этом сохраняя наложенные условия, например, ортогональности.
