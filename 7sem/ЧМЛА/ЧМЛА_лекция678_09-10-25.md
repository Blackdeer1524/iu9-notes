проблемы методы Данилевского

Проблема нулевого элемента
под диагональю 0

Случай 1
Пусть $d_{km} \ne 0, m< k-1$ тогда этот элемент ставим на место нулевого элемента $d_{k, k-1}$, те маняем нулевой элемент и ненулевой элемент местами , при этом перестановляем k-1 и m строки местами

без док-ва

$D'$ -  новая матрица полученная из D перестановкой строк и столбцов(см выше) является подобной D


Случай 2
Если матрица имеет вид
$$A = \begin{pmatrix}
D_{11} & D_{12} \\
D_{21} & D_{22}
\end{pmatrix}$$
где $D_{11}, D_{12}$ - нормальные, $D_{21}$ - нулевая, $D_{22}$ - в форме Фробениуса, тогда
$$det(D - \lambda E)= det(D_{11}-\lambda E) det(D_{22} - \lambda E)$$
, так как $D_{22}$ -фробениусова, тогда
$$det(D_{22}-\lambda E) = (-1)^{n-k-1} (\lambda^{n-k+1} - d_{kk}\lambda^{n-k} - ... - d_{kn})$$
из-за произведения и желания получить 0, мы упрощаем себе жизнь и применяем метод Данилевского только к $D_{11}$.

# Метод Крылова



$$D(\lambda) = (-1)^n(\lambda^n - p_1 \lambda^{n-1} - ... -p_n) = 0$$


Теорема Гамильтона-Кэли
Любая квадратная матрица удовлетворяет своему характеристическому уравнению. $D(A) = 0$
или
$$D(A) = A^n - p_1 A^{n-1} - ... - p_n = 0$$

Возьмем проивзольный вектор $y^{(0)} \in \mathbb R^n$, рассмотрим
$$D(\lambda) y^{(0)} = 0$$
Обозначим, что $y^{(k)} = A^k y^{(0)}$
тогда перепишем выражение
$$y^{(n)} - p_1 y^{(n-1)} - ... - p_n = 0$$

Далее из $y^{(0)}$ выбранного случайно, можем положить, что $y^{(0)} = (1, 0 ... 0 )^T$ получаем $y^{(1)} = Ay^{(0)}, y^{(2)} = Ay^{(1)} ...$

тогда получим(покоординатно)
$$\begin{cases}
p_1 y_{1}^{(n-1)} + p_2 y_{1}^{(n-2)} + ... + p_{n-1} y_{1}^{(1)} + p_n y_{1}^{(0)} = y_{1}^{(n)} \\
p_1 y_{2}^{(n-1)} + p_2 y_{2}^{(n-2)} + ... + p_{n-1} y_{2}^{(1)} + p_n y_{2}^{(0)} = y_{2}^{(n)} \\
...\\
p_1 y_{n}^{(n-1)} + p_2 y_{n}^{(n-2)} + ... + p_{n-1} y_{n}^{(1)} + p_n y_{n}^{(0)} = y_{n}^{(n)}
\end{cases}$$

Решая данное СЛАУ найдем $p_1$ ...  $p_n$ - коэфициенты характеристического уравнения. Это СЛАУ решаем любым доступным нам методом.  Получив коэфициенты СЛАУ решаем характеристическое уравнение и получаем собственные значения

Поиск собственных векторов методом Крылова
$$y^{(0)} = C_1 x^{(1)} + C_2 x^{(2)} + ... + C_n x^{(n)}$$
$x^{(i)}$ - собственные вектора с собственным значением $\lambda^{(i)}$. По определению собственных векторов
$$Ax^{(i)} = \lambda^{(i)} x^{(i)}$$
...
$$A^{k} x^{(i)} = (\lambda^{(i)})^k x^{(i)}$$
также заметим, что
$$\underbrace{Ay^{(0)}}_{y^{(1)}} = A(C_1 x^{(1)} + ... C_n x^{(n)}) = C_1 \lambda^{(1)} x^{(1)} + ... + C_n \lambda^{(n)}x^{(n)}$$
опять повторяем k раз
$$y^{(k)} = A^k y^{(0)} = ... = C_1 (\lambda^{(1)})^k x^{(1)} + ... + C_n (\lambda^{(n)})^kx^{(n)}$$

Определим полином $Q^{(i)}, i = 0 ... n-1$
$$Q^{(i)}(\lambda)  = \lambda^{n-1} + q_{1,i} \lambda^{n-2} + ... + q_{n-1, i}, \quad q_{i,j} \in \mathbb R$$
Задача выразить иксы через игрики
$$\begin{matrix}
y^{(n-1)} + q_{1i} y^{(n-2)} + ... + q_{n-1,i} y^{(0)} = (C_1 (\lambda^{(1)})^{n-1}x_1 + ... C_n (\lambda^{(n)})^{n-1}x_n) + \\
q_{1i}(C_1 (\lambda^{(1)})^{n-2}x_1 + ... + C_n (\lambda^{(n)})^{n-2}x_n) + ... + \\
q_{n-1,i}(C_1x_1 + ... + C_n x_n) = \\
= C_1((\lambda^{(1)})^{n-1} + q_{1,i} (\lambda^{(1)})^{n-2} + ... + q_{n-1,i} ) x^{(1)} + ... + C_n((\lambda^{(n)})^{n-1} + q_{1,i} (\lambda^{(n)})^{n-2} + ... + q_{n-1,i} ) x^{(n)}
\end{matrix}$$

Итого получаем
$$y^{(n-1)} + q_{1i} y^{(n-2)} + ... + q_{n-1,i} y^{(0)} = C_1 Q^{(i)}(\lambda^{(1)})x_1 + ... + C_n Q^{(i)}(\lambda^{(n)})x_n$$

Можно показать, что
$$Q^{(i)}(\lambda) = \frac{D(\lambda)}{\lambda-\lambda^{(i)}}$$
Очевидно, что(проверяется прямым перемножением)
$$Q^{(i)}(\lambda^{(j)}) = \begin{cases}
0, & i\ne j \\
D'(\lambda^{(i)}), &i = j
\end{cases}$$
Получаем СЛАУ
$$\begin{cases}
...\\
y^{(n-1)} + q_{1i} y^{(n-2)} + ... + q_{n-1,i} y^{(0)} = C_i Q^{(i)}(\lambda^{(i)})x_i\\
...
\end{cases}$$
так как собственные вектора нормируются на любую величину, получаем
$$\begin{cases}
...\\
y^{(n-1)} + q_{1i} y^{(n-2)} + ... + q_{n-1,i} y^{(0)} = x_i\\
...
\end{cases}$$
Коэффициенты $q_{ji}$ определяем по схеме Горнера:
$$q_{0i} = 1, \quad q_{ji} = \lambda^{(i)} q_{j-1,i} - p_j$$
$p_j$ - коэффициенты характеристического многочлена, которые мы уже находили.
