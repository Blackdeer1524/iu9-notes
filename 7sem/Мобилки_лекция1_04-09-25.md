## Классы в Dart
1. Поддерживаются:
	- Одиночное наследование
	- Множественное наследование интерфейсов
2. ООП модель в Dart похожа на Java и C++
3. По умолчанию любой класс в Dart есть наследник класса Object
4. У классов могут быть закрытые и открытые поля(private и public)

```Dart
class Greeter{
	var greeting; // открытое поле
	var _name; // закрытое поле
	SayHello(){
		return "${greeting} ${this._name}";
	}
	
	get name=>_name;
	set name(value)=>_name;
}

main(){
	var greeter = new Greeter();
	greeter.greeting = "Hello";
	greeter.name = "Grigory";
	greeter._name = "world"; // не работает приватность
	print("${greeter.SayHello()}");
}
```

## Неявные интерфейсы в Dart

Утверждение 1
В Dart как в Java и C++ имеются интерфейсы для их определения имеются такаие же структуры как для классов

Любой класс в Dart неявно определяет интерфейсы, включающий его открытые члены, по сути любой класс в Dart является неявно опредеенным интерфейсом или *подставным классом*
