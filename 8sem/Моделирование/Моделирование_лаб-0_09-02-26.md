# Лабораторная 0 - построение аналитической модели
Цель работы: сравнить Модель Галилея и Модель Ньютона
Почти все лабораторные для моделирования требуют творческий подход, которые по-разному дают погрешность

В моделировании другая парадигма - не продолжение курса численных методов. моделирование - открытая наука и новые модели являются открытиями. 

Далее приведены формулы для моделей.
## Модель Галилея
Постановка задачи: тело брошено под углом $\alpha$ к горизонту
$$y = \frac{-g}{2v_0^2 \cos^2 \alpha}x^2 + (\tan \alpha)x, \quad 0 < \alpha < \pi/2$$
Дает 2 решения: $x_1=0$ и $x_2$ - точка приземления после броска.
Дает более менее точные решения для малых скоростей и малых влияний окружающей среды, основано на идее равноускоренного движения.

Решение линейного уравнения
$$\frac{-g}{2v_0^2 \cos^2 \alpha}x = - (\tan \alpha),$$
$$x = \frac{2v_0^2 \cos^2\alpha \tan \alpha}{g} = \frac{v_0 \sin 2\alpha}{g}$$
## Модель Ньютона
Постановка задачи та же, только учитывает больше влияния окружения на движение. 
$$\begin{cases}
m \frac{du}{dt} = -\beta u\sqrt{u^2 + w^2}, & \frac{dx}{dt}=u \\
m \frac{dw}{dt} = -g - \beta w \sqrt{u^2 + w^2}, & \frac{dy}{dt} = w
\end{cases}$$
где $x,y$ - координаты тела, $u, w$ - составляющие скорости тела в проекции на оси $X,Y$, $\beta$ -  коэффициент сопротивления воздуха.

$$\beta \approx \frac{cs\rho}{2}$$
где $c=0.15$, $S=3$ - площадь сечения снаряда, $\rho$ - плотность воздуха

Начальный угол $\alpha = \arctan(u/w)$
