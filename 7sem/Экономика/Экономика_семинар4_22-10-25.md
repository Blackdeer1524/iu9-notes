# Рыночная экономика
4 недостатка рыночной экономики:
1. возникновение монополий - одна компания покрывает всю область рынка, не давая конкурентам возможности
2. не способен решать социальные проблемы. пенсионеры и дети не защищаются рынком.
3. резкая инфляция/дефляция
4. безработица

совпадение желаний на рынке - один хочет продать, другой хочет купить

## Индивидуальные индексы
индивидуальный индекс цены
$$I_p=\frac{p_1}{p_0}$$
$p_1$ - цена отчетного периода, $p_0$ - цена базового периода

индивидуальный индекс объема
$$I_q = \frac{q_1}{q_0}$$
$q_1$ - объем отчетного периода, $q_0$ - объем базового периода

сводный индекс товарооборота
$$I_{pq} = \frac{\sum p_{1i}q_{1i}}{\sum p_{0i} q_{0i}}$$
сводный индекс цен
$$I_p = \frac{\sum p_1 q_1}{\sum p_0 q_1}$$
сводный индекс объема
$$I_q = \frac{\sum p_0 q_1}{\sum p_0 q_0}$$

## Территориальный индекс цен
$$I_{pb/a} = \frac{\sum p_b Q}{\sum p_a Q}$$
$$Q = q_a + q_b$$
индекс цен региона b относительно региона a. Если 2 региона входят в какой-то общий регион, то можно брать общее производство товара в этом регионе

Есть еще территориальный индекс объема
$$I_{qb/a} = \frac{\sum \overline P q_b}{\sum \overline P q_a} \quad \overline P = \frac{\sum p_iq_i}{\sum q_i}$$
